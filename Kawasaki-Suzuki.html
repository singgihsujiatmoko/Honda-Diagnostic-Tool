Kawasaki ECU Diagnostic

Protokol KDS (Kawasaki Diagnostic System) menggunakan ISO 14230 KWP2000

Init Wake Up (Inisialisasi)

Fast Init (5 Baud Init)

// Wake Up Pattern

Baud Rate: 10400 bps

Init Address: 0x33

Wake Up Pulse: 25ms LOW, 25ms HIGH

TX: 81 11 F1 81 04

RX: 83 F1 11 C1 8F EF 9F

Slow Init Sequence

// 5 Baud Address

Address Byte: 0x33 (200ms per bit)

Key Bytes: 0x08 0x08

Inverted Key: ~KeyByte2

Sync: 0x55

Response: 55 08 08

Langkah-langkah Wake Up:

  1. Koneksi K-Line ke pin diagnostik (biasanya pin 4)
  2. Kirim wake up pulse: LOW 25ms â†’ HIGH 25ms
  3. Kirim Start Communication Request (0x81)
  4. Tunggu Positive Response dari ECU
  5. ECU siap menerima perintah diagnostic

Suzuki ECU Diagnostic

Protokol SDS (Suzuki Diagnostic System) menggunakan ISO 14230 KWP2000 / ISO 9141-2

Init Wake Up (Inisialisasi)

Fast Init Protocol

// Suzuki Fast Init

Baud Rate: 10400 bps

Init Address: 0x33

Wake Pattern: 25ms LOW, 25ms HIGH

TX: 81 12 F1 81 05

RX: 83 F1 12 C1 EA 8F C0

Alternative Init (ISO 9141)

// 5 Baud Slow Init

Address: 0x33 @ 5 baud

Sync Pattern: 0x55

Key Bytes: 0xEA 0x8F

Response: 55 EA 8F

Inverted: ~0x8F = 0x70

Langkah-langkah Wake Up:

  1. Hubungkan ke dealer mode connector (pin K-Line)
  2. Kirim inisialisasi: 25ms LOW pulse diikuti 25ms HIGH
  3. Kirim Start Communication: 81 12 F1 81 [checksum]
  4. Tunggu response positif dari ECU (C1 xx xx)
  5. Session diagnostic aktif, siap kirim command

Frame Format KWP2000

Format Header

[FMT] Format Byte

[TGT] Target Address

[SRC] Source Address

[LEN] Length (if applicable)

[SID] Service ID

[DATA] Data Bytes

[CS] Checksum

Checksum Calculation

// Simple 8-bit sum

CS = (FMT + TGT + SRC + DATA...) & 0xFF

// Example:

81 + 11 + F1 + 81 = 204

204 & 0xFF = 0x04
